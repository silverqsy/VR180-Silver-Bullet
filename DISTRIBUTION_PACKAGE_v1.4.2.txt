   - Arrow buttons now increment by 0.1° (was 0.001°)
   - Makes adjustments 100x faster

   - Affects Global Panomap and Stereo Offset controls
   - Status: ✅ IMPROVED

🚀 INSTALLATION INSTRUCTIONS
================================================================================

WINDOWS (Build from Source):
  1. Extract VR180_Silver_Bullet_v1.4.2_Windows_BuildKit.zip
  2. Install prerequisites:
     • Python 3.10+ (add to PATH)
     • FFmpeg full build (add to PATH)
  3. Run build_windows.bat
  4. Find app in: dist\VR180 Silver Bullet\
  5. Run: VR180Processor.exe

  See README.md in BuildKit for detailed instructions

macOS (Ready to Run):
  1. Extract VR180_Silver_Bullet_v1.4.2_macOS.zip
  2. Drag "VR180 Silver Bullet.app" to Applications folder
  3. Right-click app → Open (first time only, for Gatekeeper)
  4. App is ready to use!

  Requirements:
  • macOS 10.15 (Catalina) or newer
  • Apple Silicon or Intel Mac
  • 8 GB RAM minimum (16 GB recommended)

⚠️ KNOWN ISSUES
================================================================================

WINDOWS:
  • Hardware decode in preview may crash on some systems (disabled by default)
  • Some NVIDIA GPUs require latest drivers for NVENC
  • ProRes encoding requires Windows 10 or newer

macOS:
  • First launch requires right-click → Open (Gatekeeper)
  • MV-HEVC requires external spatial-media-kit-tool
  • Some older Macs don't support VideoToolbox 10-bit encoding
  • App is not notarized (no developer account)

ALL PLATFORMS:
  • Very large files (>100GB) may cause memory issues
  • Some fisheye lens types not yet supported
  • LUT must be in .cube format
