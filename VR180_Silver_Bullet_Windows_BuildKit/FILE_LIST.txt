ESSENTIAL FILES (Required for Building)
========================================

vr180_gui.py
  - Main application source code
  - Contains all GUI, processing, and preview logic
  - Updated with new defaults (200 Mbps bitrate, bitrate mode)
  - Smart 10-bit input handling

vr180_processor.spec
  - PyInstaller build specification
  - Defines how to bundle the application
  - Includes FFmpeg, spatialmedia, and all dependencies
  - Fixed icon path (icon.ico)

requirements.txt
  - Python package dependencies
  - PyQt6 >= 6.4.0
  - numpy >= 1.21.0
  - pyinstaller >= 6.0.0

icon.ico
  - Application icon for Windows
  - 256x256 pixel icon
  - Displayed in taskbar and window title

spatialmedia/
  - Google's spatial media library (modified)
  - Used for VR180 metadata injection
  - Required for YouTube VR180 compatibility
  - Directory structure preserved in build


BUILD SCRIPTS
========================================

build_windows.bat
  - Automated build script (Batch file)
  - Double-click to build
  - Checks for Python and FFmpeg
  - Installs dependencies automatically
  - Verifies bundled files
  - Most user-friendly option

build_windows.ps1
  - PowerShell build script
  - More detailed output and error checking
  - Color-coded status messages
  - Run in PowerShell: .\build_windows.ps1
  - Alternative to batch file

check_requirements.bat
  - Validates build prerequisites
  - Checks Python installation
  - Checks FFmpeg installation (full build)
  - Checks FFprobe
  - Lists installed Python packages
  - Run before building to ensure everything is ready

BUILD OUTPUT (After Building)
========================================

After running build_windows.bat, you'll get:

build/
  - Temporary build files
  - Can be deleted after successful build
  - Auto-cleaned on next build

dist/
  - Final application output
  - This is what you distribute

dist/VR180 Silver Bullet/
  - Standalone application folder
  - Contains everything needed to run
  - No installation required
  - Size: ~250-300 MB

dist/VR180 Silver Bullet/VR180Processor.exe
  - Main executable (double-click to run)
  - Size: ~35-50 MB

dist/VR180 Silver Bullet/ffmpeg.exe
  - Bundled FFmpeg binary
  - Used for video processing
  - Size: ~100-150 MB

dist/VR180 Silver Bullet/ffprobe.exe
  - Bundled FFprobe binary
  - Used for video analysis
  - Size: ~10-20 MB

dist/VR180 Silver Bullet/spatialmedia/
  - VR180 metadata module
  - Required for metadata injection

dist/VR180 Silver Bullet/_internal/
  - Python runtime and libraries
  - PyQt6 DLLs
  - NumPy libraries
  - Other dependencies
  - Size: ~100-150 MB
