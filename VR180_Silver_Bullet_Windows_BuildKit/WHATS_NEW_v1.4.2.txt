  VR180 Silver Bullet v1.4.2 - What's New

RELEASE DATE: January 18, 2026

--------------------------------------------------------------------------------
🐛 BUG FIXES
--------------------------------------------------------------------------------

1. DRAG AND DROP CRASH FIXED (macOS)
   - Fixed fatal crash when dragging video files into application
   - App is now stable with drag-and-drop workflow
   - Added error handling to prevent future crashes

2. DRAG AND DROP PREVIEW LOADING
   - Dropping video files now automatically loads preview
   - Fixed issue where preview stayed blank after drop
   - Works identically to Browse button

3. SPINBOX STEP SIZE IMPROVED
   - Arrow buttons now increment by 0.1° instead of 0.001°
   - Makes adjustments 100x faster
   - Affects Global Panomap and Stereo Offset controls
   - Still shows 3 decimal precision for fine control

--------------------------------------------------------------------------------
📋 UPGRADE NOTES
--------------------------------------------------------------------------------

WINDOWS:
  1. Extract VR180_Silver_Bullet_v1.4.2_Windows.zip
  2. Run VR180Processor.exe
  3. Settings from v1.4.1 are preserved automatically

macOS:
  1. Open VR180_Silver_Bullet_v1.4.2_macOS.dmg
  2. Drag to Applications (replace if upgrading)
  3. Right-click → Open on first launch (Gatekeeper)
