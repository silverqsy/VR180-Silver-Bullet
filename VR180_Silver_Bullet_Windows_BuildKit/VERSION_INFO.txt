BUILD REQUIREMENTS
--------------------------------------------------------------------------------

REQUIRED:
  • Python 3.10 or newer (with pip)
  • FFmpeg full build (with all codecs)
  • PyInstaller 6.0+
  • PyQt6
  • NumPy
  • OpenCV (cv2)

OPTIONAL:
  • spatial-media-kit-tool (for MV-HEVC support)

INSTALLATION:
  1. Install Python 3.10+ (add to PATH)
  2. Install FFmpeg (add to PATH)
  3. Run: build_windows.bat
